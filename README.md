# Nexus AI — Premium Landing Page

A luxury, conversion-focused website for **Nexus AI**, an intelligent automation platform.

## Design
- **Apple × Stripe × Notion** level polish
- Purple-forward palette with soft off-white backgrounds
- Poppins + Inter typography, smooth animations, responsive layout

## Sections

## Tech Stack
- Semantic HTML5
- Modern CSS (Flexbox/Grid, custom properties, keyframe animations)
- Vanilla JavaScript (IntersectionObserver reveals, tab switching, counter animations, 3D tilt)

## Run Locally
Open `index.html` in your browser — no build step required.
